Historical Weather Data API

Search for a particular date and station: http://127.0.0.1:8000/v1/api/{station ID}/{date: YYYY-MM-DD}
Example: http://127.0.0.1:8000/v1/api/10/1992-05-19

Search for a particular year (or plus month) and station: http://127.0.0.1:8000/api/v1/yearly/{station ID}/{year: YYYY}
Example: http://127.0.0.1:8000/v1/api/yearly/10/1992
http://127.0.0.1:8000/v1/api/yearly/10/1992-10

Search for a particular station all years: http://127.0.0.1:8000/v1/api/{station ID}
Example: http://127.0.0.1:8000/v1/api/10

If Temperature(TG) is "-999.9" it means the data(temperature) has lost !
